# 📊 HR Attrition Dashboard (Excel)

## 📌 About This Project
This is a **learning project** where I built an HR Attrition Dashboard using Microsoft Excel.  
The goal was to understand how to analyze data and create interactive dashboards.

---

## 🔍 What I Practiced
- Creating Pivot Tables and Charts  
- Using Power Pivot (Data Model)  
- Building relationships between tables  
- Using slicers for interactivity  
- Designing a simple dashboard layout  

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables & Charts  
- Power Pivot  
- Slicers
